# About Section Images

## Caricature Images

This directory contains the AI-generated caricature images used in the About section.

### Current Files
- `milan-caricature@1x.webp` (420x420px) - Standard resolution for regular displays
- `milan-caricature@2x.webp` (800x800px) - High resolution for Retina/HiDPI displays

### Note
The current images are placeholders created from the profile photo. Please replace these with the actual AI-generated caricature images when available.

To replace the images:
1. Convert the new caricature images to WebP format
2. Create two versions:
   - 1x: 420px width (for standard displays)
   - 2x: 840px width (for high-DPI displays)
3. Replace the existing files with the same names
4. Commit and push the changes

### Image Requirements
- Format: WebP (for optimal compression and quality)
- Dimensions: 
  - 1x: 420px width (maintains aspect ratio)
  - 2x: 840px width (maintains aspect ratio)
- Quality: 85% (good balance between file size and quality)
- Alt text: The alt text in index.html mentions this is an AI-generated illustration
